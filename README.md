# CoreQuest

> Understanding computer systems from silicon to software.

---

## About

CoreQuest is a long-term engineering project created to document my journey of learning how computer systems work.

Instead of simply learning how to use technologies, the goal is to understand the principles behind them—from hardware and embedded systems to operating systems, networking, firmware, reverse engineering, and cybersecurity.

This repository serves as the single source of truth for my learning journey.

---

## Mission

To understand computer systems from silicon to software.

---

## Learning Philosophy

- Learn by understanding, not by memorization.
- Document everything.
- Every question is worth asking.
- Build knowledge layer by layer.
- Curiosity is more important than speed.
