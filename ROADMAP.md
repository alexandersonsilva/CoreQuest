# CoreQuest Roadmap

> A roadmap organized by layers of computer systems rather than by technologies.

---

# Era 0 — Foundations

Understand the fundamental concepts that every computer system is built upon.

## Objectives

- Binary and hexadecimal
- Boolean logic
- Computer architecture
- Operating system fundamentals
- Linux basics
- Git and GitHub
- Networking fundamentals
- Programming fundamentals

---

# Era 1 — Hardware

Understand how a computer exists physically.

## Objectives

- Electronic components
- Printed Circuit Boards (PCB)
- CPUs and SoCs
- RAM and Flash memory
- Power systems
- Communication buses
- UART
- SPI
- I²C
- GPIO

---

# Era 2 — Embedded Systems

Understand small computers and dedicated devices.

## Objectives

- Boot process
- Bootloader
- Device Tree
- Embedded Linux
- Build systems
- Cross-compilation

---

# Era 3 — Firmware

Understand software that interacts directly with hardware.

## Objectives

- Firmware architecture
- Memory mapping
- Flash layout
- Device drivers
- U-Boot
- Kernel basics

---

# Era 4 — Reverse Engineering

Understand systems without access to their original design.

## Objectives

- Static analysis
- Dynamic analysis
- Ghidra
- Binary analysis
- Firmware extraction
- Debugging

---

# Era 5 — Cybersecurity

Understand how to evaluate and improve system security.

## Objectives

- Secure boot
- Firmware analysis
- Vulnerability research
- Exploit mitigation
- Embedded security
- Offensive security

---

# Continuous Skills

These skills evolve throughout the entire project.

- Documentation
- Technical writing
